# Restaurant Management & Table Booking System
## Introduction

This project is a web-based Restaurant Management System that allows customers to explore menus, book tables online, and receive promotions, while enabling restaurant managers to efficiently manage operations such as bookings, menus, events, and customer feedback.

The system is designed to enhance user experience with a modern interface and provide a seamless workflow for both customers and administrators.

## Features
### Customer Side
View Menu & Restaurant Information
- Browse detailed menu:
  - Dish name
  - Image
  - Description
  - Price
  - Category (Main dish, Appetizer, Vegetarian, Drinks, etc.)
  - View restaurant information:
  - Location
  - Space & ambiance
  - Opening hours
  - Chef introduction
  - Special events
- Online Table Booking
- Book tables by selecting:
  - Date & time
  - Number of guests
  - Customer information
  - Special occasions
  - Real-time availability checking
  - Multiple table types:
  - Standard
  - Premium
- Subscribe for Promotions
- Customers can subscribe via email to receive promotions and updates
  - Rating & Feedback
  - Leave ratings using star system
  - Provide detailed feedback about services
### Admin Side
- Booking Management
  - Create / Update / Delete bookings
  - Confirm customer arrival and record total bill
  - Real-time booking list:
  - Time
  - Customer name
  - Number of guests
  - Notes
  - Filter by:
  - Date
  - Status
  - Table type
  - Guest count
  - Auto-cancel bookings if customer does not arrive after 30 minutes
- Menu Management
  - Add / Edit / Delete dishes
  - Upload images and descriptions
  - Highlight best-selling dishes on homepage
- Event Management
  - Manage events displayed on the website
- Statistics & Reports
- View reports by:
  - Daily / Monthly bookings
  - Customer count
  - Revenue
- Review Management
  - View all customer reviews
  - Remove inappropriate reviews
- Email Marketing
  - Send promotional emails to subscribed customers
- Technologies Used
  - Frontend: ReactJS, Tailwind
  - Backend: NodeJS, ExpressJS
  - Database: MySQL
- Authentication: JWT
- Project Goals
  - Improve user experience in restaurant booking
  - Provide efficient management tools for restaurant owners
  - Practice full-stack development with real-world scenarios
