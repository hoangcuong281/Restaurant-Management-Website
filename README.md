# TokyoBites

Dự án trang web quản lý nhà hàng được xây dựng để mô phỏng giao diện và chức năng cơ bản của một hệ thống quản lý nhà hàng hiện đại. Bao gồm trang landing page giới thiệu về nhà hàn và trang admin quản lý thông tin người dùng. Dự án có cấu trúc frontend (ReactJS) + backend (NodeJS, MySQL) rõ ràng, cho phép mở rộng các tính năng quản lý như hiển thị thực đơn, đặt bàn, và xử lý các tương tác với người dùng.
